怎么行呢？稿子太差，差

得不上路，要想发表，只有

“斧正”。于是，亲自动手

，“妙手回春”了一个晚上

，总算有了个样子。第二天

一早，又亲自送到报社，为

保证采用，送到的同时又给

相关编辑送了他半个月的工

资。后来，军区报纸当要闻

在头版上隆重刊登了，科长

的大名尊姓跃然其上，前无

阿今之名，后无阿今之姓，

纯粹系科长之“独著”。
这是科长第一回看见自个名

字赫然上报，简直有些乐不

可支，有些忘乎所以，捧着

报纸先是给政委看，受了大

表扬；回家又给爱人看，孩

子看，又受了大表扬；还寄

给好多朋友看，完全失去了

平时“含而不露”的作风，

表现出了常人都有的那种首

次见稿的喜悦。对稿子是经

过怎样秘密的过程发表出来

、发出来的有多少文字属于

他自个，等等细节实情却是

只字不提。对阿今不签名也

没表示异议，好像阿今帮他

改稿送稿什么的都是应该
